AlgoTrack

AlgoTrack is a coding practice and problem-solving platform inspired by LeetCode.
It helps users practice DSA questions, track progress, and improve problem-solving skills
through a structured and interactive environment.


Features
- User Authentication (Login / Register)
- Curated DSA coding problems
- Difficulty-based categorization (Easy / Medium / Hard)
- Topic-wise problems (Arrays, Strings, DP, Graphs, etc.)
- Solve and track problems
- Progress tracking dashboard

  
Tech Stack
Frontend: React (Vite), HTML, CSS, JavaScript
Backend: Node.js, Express.js
Database: MongoDB Atlas
